# Node.js Mastery Playground

A curated collection of **Node.js practice projects** covering core backend concepts, Express.js APIs, MongoDB integration, authentication, file uploads, streams, clustering, sessions, and URL shortening.

This repository is designed as a hands-on learning workspace for building and understanding real-world backend patterns with Node.js.

---

## Highlights

- **Core Node.js concepts** using the built-in `http`, `fs`, and `cluster` modules
- **Express.js REST APIs** with routing, request handling, and JSON responses
- **MongoDB + Mongoose** examples for data modeling and persistence
- **JWT authentication** with protected routes and role-based access control
- **File upload handling** with Multer
- **Streaming and compression** for efficient file delivery
- **URL shortener service** with visit tracking
- **Session-based and MVC-style patterns** for backend architecture practice
